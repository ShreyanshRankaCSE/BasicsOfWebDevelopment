Let's learn Arrays Today

Definition: In Javascript, Arrays are treated as objects which can store values in it.

In Arrays, values can be entered in strings, numbers, etc. :

  1. Array of Strings
  2. Array of Numbers
  3. Empty Array: Returns Empty Array
  4. Mixed Datatypes of Array : We can store values in array by mixing different types of Datatypes

Values in Arrays are called by the indexes present (Starting from 0)

Actions are performed on the Array

  1. Push: Pushes Data at the end of the array
  2. Unshift: Adds the data at the starting of the array
  3. Pop: Removes the last element from the array
  4. Shift: Removes the first element from the array
  5. Length: Gets the total elements present in the Array i.e. Length of Array
  6. Concat: Combines 2 or more Arrays and returns it.
